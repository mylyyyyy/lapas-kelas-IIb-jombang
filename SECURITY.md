# Security Policy

We take the security of this system seriously. Given the nature of this application (Correctional Institution Management System / Sistem Lapas), ensuring data integrity and privacy is our top priority.

## Supported Versions

Only the latest stable version is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **DO NOT** create a public issue on GitHub. Publicly disclosing a vulnerability can put the system at risk before we have a chance to patch it.

### How to Report
Please send a detailed report via email to the maintainer:

* **Email:** aryadian003@gmail.com
* **Subject:** [SECURITY REPORT] - Lapas Jombang System

### What to Include
In your email, please include:
1.  A description of the vulnerability.
2.  Steps to reproduce the issue (Proof of Concept).
3.  Potential impact of the vulnerability.

### Our Response
* We will acknowledge receipt of your report within **48 hours**.
* We will provide an estimated timeframe for the fix.
* Once fixed, we will notify you and release a patch.

Thank you for helping us keep the system safe.
